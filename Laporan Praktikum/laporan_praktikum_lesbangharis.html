BAB II PEMBAHASAN


2.1. SEJARAH HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menampilkan dokumen web di internet. HTML menggunakan tanda-tanda atau markup untuk mengatur teks dan media lainnya, seperti gambar dan video, di dalam dokumen web. HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1989 saat bekerja di CERN (Organisasi Eropa untuk Penelitian Nuklir). Ia menciptakan HTML untuk membuat sistem informasi yang terdistribusi di internet yang dapat diakses oleh seluruh pengguna di seluruh dunia. Versi pertama dari HTML, yang disebut HTML 1.0, diperkenalkan pada tahun 1991. Sejak saat itu, HTML telah mengalami banyak peningkatan dan perkembangan, dan sekarang telah mencapai versi HTML 5.0. HTML saat ini merupakan standar industri untuk membuat halaman web, dan banyak perangkat lunak yang dapat digunakan untuk menulis kode HTML, seperti teks editor dan aplikasi pengembangan web seperti Adobe Dreamweaver.

2.3. ELEMEN-ELEMEN PADA HTML

Elemen HTML adalah bagian dari dokumen HTML yang memiliki makna tertentu dan ditandai dengan tag-tag yang menandai awal dan akhir elemen tersebut. Elemen HTML dapat berisi teks, gambar, video, dan elemen lainnya, sesuai dengan makna yang dimilikinya.
Berikut ini adalah beberapa contoh elemen HTML yang sering digunakan:

  • '<p>' (paragraf) :

    • Merupakan elemen yang digunakan untuk membuat paragraf teks.

  • '<h1>' sampai '<h6>' (heading) :

    • Merupakan elemen yang digunakan untuk membuat heading atau judul. Tingkatan heading ditentukan oleh angka yang terdapat dalam tag, dimana '<h1>' adalah heading level 1 (tertinggi) dan '<h6>' adalah heading level 6 (terendah).

  • '<img>' (image) :

    • Merupakan elemen yang digunakan untuk memasukkan gambar ke dalam dokumen HTML.

  • '<a>' (anchor) :

    • Merupakan elemen yang digunakan untuk membuat link ke halaman web atau dokumen lain.

  • '<div>' (division) :

    • Merupakan elemen yang digunakan untuk membagi dokumen HTML menjadi bagian-bagian yang lebih kecil.

  • '<form>' :

    • Merupakan elemen yang digunakan untuk membuat formulir di halaman web.

  • '<style>' :

    • Merupakan elemen yang digunakan untuk menentukan gaya atau tampilan elemen-elemen di dalam dokumen HTML. Elemen '<style>' biasanya ditempatkan di bagian kepala dokumen '<head>' dan menggunakan bahasa CSS (Cascading Style Sheets) untuk menentukan gaya.     2.4. TABEL PADA HTML

    Tabel adalah elemen HTML yang digunakan untuk menampilkan data dalam bentuk baris dan kolom. Tabel terdiri dari baris-baris (row) yang terdiri dari sel-sel (cell) yang masing-masing berisi data tertentu. Untuk membuat tabel di HTML, Anda dapat menggunakan tag '<table>' yang menandai awal dan akhir tabel. Tag '<table>' harus diisi dengan elemen-elemen lain yang menyusun tabel, seperti '<tr>' (table row) yang menandai baris tabel, dan '<tr>' (table cell) yang menandai sel tabel.     BAB III PENUTUP


    3.1. KESIMPULAN

    HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menampilkan dokumen web di internet. HTML menggunakan tanda-tanda atau markup untuk mengatur teks dan media lainnya, seperti gambar dan video, di dalam dokumen web. Struktur dasar dokumen HTML terdiri dari element '<html>', element '<head>', element '<title>', dan element '<body>'. Element '<html>' merupakan element yang menandai awal dan akhir dokumen HTML, element '<head>' merupakan element yang menandai awal dan akhir bagian kepala dokumen HTML, element '<title>' merupakan element yang menandai awal dan akhir judul dokumen, dan element '<body>' merupakan element yang menandai awal dan akhir bagian tubuh dokumen HTML. Elemen-elemen pada HTML meliputi '<p>' (paragraf), '<h1>' sampai '<h6>' (heading), '<img>' (image), '<a>' (link), dan lain-lain. HTML juga memiliki atribut yang dapat digunakan untuk memberikan informasi tambahan kepada elemen, seperti atribut 'src' pada elemen '<img>' yang digunakan untuk menentukan lokasi file gambar yang akan ditampilkan. HTML saat ini merupakan standar industri untuk membuat halaman web dan banyak digunakan oleh pengembang web di seluruh dunia.
